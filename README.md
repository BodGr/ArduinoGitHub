# ArduinoGitHub

Dies ist ein Lernprojekt, um Git und GitHub in Verbindung mit Arduino und ESP32 zu üben.

## Ziel

- Git-Grundlagen lernen: commit, push, pull, branches, merge
- Sicher experimentieren, ohne produktive Projekte zu gefährden
- Kleine ESP32-Sketches versionieren und dokumentieren
